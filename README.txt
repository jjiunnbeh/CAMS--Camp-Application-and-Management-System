Group 1 CAMsAPP

This project was developed and tested using Eclipse IDE, JAVA-19. 

Inside the SCSE_grp1 folder there will be folders for the report, diagrams and source code


Report and Declarations Folder:
Contains the assignment report (SCSE-grp1.pdf) and 4 signed declarations.

Diagrams Folder:
Contains the UML Class diagram, Architecture Diagram, State Machine Diagram

SCSE_grp1 folder: 
src - contains the source code
doc - javadoc


To run the application, go to src->camsApp-> CAMsApp and run as 1 java application.
